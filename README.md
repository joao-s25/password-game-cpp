# 🔒 Password Game (C++)
---
### Jogo simples feito em C++ onde o jogador precisa adivinhar uma senha de 5 dígitos gerada aleatoriamente.

---
## 💻 Como funciona

- O programa gera uma senha aleatória de cinco dígitos
- O jogador tem até 20 rodadas para acertar
- Após cada tentativa, o jogo informa quantos números estão corretos, e quantos estão nas posições corretas
- O jogador vence ao acertar todos os dígitos em suas posições corretas

---
## 🌎 Idiomas

Atualmente o jogo possui suporte para:

- 🇧🇷 Português (pt)
- 🇺🇸 Inglês (en)

---
## ▶ Como executar

### Usando Dev-C++
-  Abra o arquivo `PasswordGame.dev`
- Clique em Compile & Run

### Usando terminal (g++)

```bash
g++ main.cpp -o game
./game
```
